## Overview

EduMate allows you to export generated assessments as professionally formatted Microsoft Word documents (.docx). This enables you to distribute assessments to students, print them, or integrate them into your learning management system.

## Export Formats

EduMate provides two distinct document types:

<CardGroup cols={2}>
  <Card title="Question Paper" icon="file-lines" color="#6366f1">
    Clean assessment for student distribution with questions and answer options only—no solutions.
  </Card>
  
  <Card title="Answer Key" icon="key" color="#10b981">
    Complete solution guide for instructors with correct answers and explanations highlighted.
  </Card>
</CardGroup>

<Warning>
  Always verify you're exporting the correct format. Accidentally distributing the Answer Key to students will compromise the assessment.
</Warning>

## Exporting from the Results Screen

Immediately after generating an assessment:

### Exporting Question Paper

<Steps>
  <Step title="Switch to Question View">
    Ensure you're in "Question Paper" mode by clicking the **"Question Paper"** tab at the top of the assessment viewer (styled in violet/purple).
  </Step>
  
  <Step title="Click Download Button">
    Click the **"Download Questions (.docx)"** button in the sticky header.
    
    <Info>
      While downloading, the button shows a loading spinner and text changes to "Generating..."
    </Info>
  </Step>
  
  <Step title="Save File">
    Your browser downloads the file automatically:
    - **Filename**: `EduMate_Question_Paper.docx`
    - **Format**: Microsoft Word 2016+ compatible
    - **Location**: Your default downloads folder
  </Step>
</Steps>

### Exporting Answer Key

<Steps>
  <Step title="Switch to Answer View">
    Click the **"Answer Key"** tab at the top of the assessment viewer (styled in emerald/green).
  </Step>
  
  <Step title="Click Download Button">
    Click the **"Download Answer Key (.docx)"** button in the sticky header.
    
    <Tip>
      Notice the button color matches the view mode—green for Answer Key, purple for Question Paper.
    </Tip>
  </Step>
  
  <Step title="Save File">
    The file downloads automatically:
    - **Filename**: `EduMate_Answer_Key.docx`
    - **Format**: Microsoft Word 2016+ compatible
    - **Confidential**: For instructor use only
  </Step>
</Steps>

## Exporting from History

To export a previously generated assessment:

<Steps>
  <Step title="Navigate to History">
    Click **"History"** in the sidebar to view all saved assessments.
  </Step>
  
  <Step title="Quick Download (Question Paper Only)">
    Click the **"Download"** button next to any assessment in the table.
    
    <Note>
      The history table download button exports only the Question Paper. For Answer Keys, use the View modal (next step).
    </Note>
  </Step>
  
  <Step title="View Assessment for Full Export Options">
    Alternatively:
    1. Click **"View"** to open the assessment in the slide-over modal
    2. Use the in-viewer download buttons to export either Question Paper or Answer Key
  </Step>
</Steps>

## Document Structure

### Question Paper Format

The exported Question Paper includes:

**Header Section**:
- **Title bar**: "EduMate" branding in indigo
- **Subtitle**: "Question Paper | Total Questions: [count] | Generated by EduMate AI"

**Page Header** (every page):
- "EduMate AI Assessment — Question Paper" in light gray

**Page Footer** (every page):
- Centered page numbers ("Page X of Y")

**Question Format** (for each MCQ):
- Question number in shaded indigo box (e.g., "Q1")
- Question text in clear, readable font
- Answer options labeled A, B, C, D (or more if applicable)
- Consistent spacing and professional layout
- Horizontal rules between questions

<Tip>
  The Question Paper format is designed for direct printing or digital distribution—no additional formatting required.
</Tip>

### Answer Key Format

The exported Answer Key includes everything in the Question Paper, plus:

**Enhanced Question Headers**:
- Question number in shaded emerald/green box (color-coded as "solution")

**Highlighted Correct Answers**:
- Correct option highlighted with green background
- Checkmark symbol (✓) before the letter
- Bold text formatting

**Answer Summary**:
- "Correct Answer: [letter]" displayed prominently below options
- Underlined for emphasis

**Explanations**:
- Blue-shaded explanation box after each question
- "Explanation:" label in bold
- AI-generated reasoning for why the answer is correct

**Document Marking**:
- Page header: "EduMate AI Assessment — Answer Key (CONFIDENTIAL)"
- Subtitle: "Answer Key | Total Questions: [count] | For Professor Use Only"

<Warning>
  The Answer Key is clearly marked as CONFIDENTIAL in the header to prevent accidental distribution to students.
</Warning>

## Document Specifications

### Technical Details

| Aspect | Specification |
|--------|---------------|
| **File Format** | DOCX (Office Open XML) |
| **Compatibility** | Microsoft Word 2016+, Google Docs, LibreOffice |
| **Font** | Calibri (system default) |
| **Encoding** | UTF-8 |
| **Page Size** | US Letter (8.5" × 11") |
| **Margins** | Standard (1" all sides) |

### Styling Features

- **Color-coded headers**: Indigo for questions, emerald for answer keys
- **Professional shading**: Light background fills for headers and highlights
- **Consistent typography**: Balanced font sizes for readability
- **Borders and rules**: Subtle dividers between questions
- **Page numbering**: Automatic pagination in footer

## Editing Downloaded Documents

After downloading, you can:

<Accordion title="Edit in Microsoft Word">
  1. Open the `.docx` file in Microsoft Word
  2. Modify question text, options, or formatting as needed
  3. Add school/course branding
  4. Insert instructions or additional sections
  5. Save changes normally
  
  <Note>
    All formatting (colors, fonts, tables) is editable in Word.
  </Note>
</Accordion>

<Accordion title="Convert to PDF">
  For distribution, convert to PDF:
  
  **In Microsoft Word**:
  - File → Save As → PDF
  
  **In Google Docs**:
  - File → Download → PDF Document
  
  **Online Converters**:
  - CloudConvert, Adobe online tools, etc.
</Accordion>

<Accordion title="Print Directly">
  The documents are print-ready:
  1. Open the DOCX file
  2. File → Print
  3. Select printer and adjust print settings
  4. Print
  
  <Tip>
    For large quantities, consider duplex (double-sided) printing to save paper.
  </Tip>
</Accordion>

<Accordion title="Import to LMS">
  Many Learning Management Systems accept Word documents:
  
  - **Canvas**: Upload to Assignments or Quizzes module
  - **Blackboard**: Add to Course Materials
  - **Moodle**: Upload as Resource
  - **Google Classroom**: Attach to Assignment
  
  <Info>
    Some LMS platforms can automatically parse MCQ content from DOCX files into their quiz engines.
  </Info>
</Accordion>

## Behind the Scenes: DOCX Generation

EduMate uses the `docx` JavaScript library to programmatically build Word documents:

1. **Document Structure**: Creates sections with headers and footers
2. **Paragraph Styling**: Applies shading, colors, and formatting
3. **Content Population**: Inserts questions, options, and explanations
4. **Binary Conversion**: Packages everything into a valid DOCX file
5. **Browser Download**: Triggers file save using FileSaver.js

<Note>
  This process happens entirely in your browser—no server-side conversion required. Downloads are instant.
</Note>

## Troubleshooting Export Issues

<AccordionGroup>
  <Accordion title="Download button stays disabled">
    **Cause**: Assessment data not fully loaded.
    
    **Solution**:
    - Wait a few seconds for the viewer to fully render
    - Refresh the page and try again
    - Ensure you're viewing a completed assessment
  </Accordion>
  
  <Accordion title="File doesn't download">
    **Check**:
    - Browser pop-up blocker settings (allow downloads from EduMate)
    - Browser download permissions
    - Available disk space
    - Antivirus software blocking downloads
    
    **Try**:
    - Right-click the download button and select "Save link as..."
    - Use a different browser
    - Clear browser cache and cookies
  </Accordion>
  
  <Accordion title="Downloaded file won't open">
    **Possible causes**:
    - Corrupted download
    - Unsupported software version
    - File extension changed
    
    **Solutions**:
    - Re-download the file
    - Ensure filename ends in `.docx`
    - Open with Microsoft Word, Google Docs, or LibreOffice (not older Office 2003)
    - Try opening in a different application
  </Accordion>
  
  <Accordion title="Formatting looks wrong when opened">
    **Cause**: Font or style incompatibility.
    
    **Fix**:
    - Install Calibri font (default in Windows, available for macOS/Linux)
    - Accept "Update styles" if prompted by Word
    - Manually adjust formatting if necessary
    
    <Tip>
      Google Docs may display slightly different fonts but maintains the overall layout.
    </Tip>
  </Accordion>
  
  <Accordion title="Special characters display incorrectly">
    **Cause**: Encoding issue or missing Unicode support.
    
    **Solution**:
    - Ensure you're using a modern version of Word (2016+)
    - Try opening in Google Docs (better Unicode support)
    - Check that your system has necessary fonts installed
  </Accordion>
</AccordionGroup>

## Best Practices

<CardGroup cols={2}>
  <Card title="Separate Storage" icon="folder-tree">
    Keep Question Papers and Answer Keys in separate folders to prevent accidental distribution of solutions.
  </Card>
  
  <Card title="Version Control" icon="clock">
    Include dates in filenames when saving multiple versions: `EduMate_Question_Paper_2026-01-15.docx`
  </Card>
  
  <Card title="Backup Copies" icon="copy">
    Save backup copies to cloud storage (Google Drive, OneDrive) for safety.
  </Card>
  
  <Card title="Review Before Distributing" icon="clipboard-check">
    Always open and review the exported document before sharing with students to ensure quality.
  </Card>
</CardGroup>

## Security Considerations

<Warning>
  **Protect Your Answer Keys**
  - Store Answer Keys in password-protected folders
  - Never email Answer Keys to students
  - Delete Answer Keys from shared drives
  - Mark files as "CONFIDENTIAL" or "INSTRUCTOR ONLY"
</Warning>

<Info>
  Question Papers are safe to distribute publicly as they contain no solutions or confidential information.
</Info>
